Vision-AR

Plataforma de monitoreo centralizado de infraestructura tecnológica.

Descripción

Vision-AR es una plataforma web desarrollada para centralizar la supervisión de cámaras, routers, switches, servidores y otros dispositivos desde una sola interfaz.

Problema

La supervisión manual de dispositivos distribuidos dificulta detectar fallas rápidamente y centralizar la información operativa.

Solución

Desarrollo de una plataforma web que concentra el estado de los dispositivos, automatiza validaciones y facilita la gestión de usuarios, permisos y alertas.

Mi participación

  • Backend con Python y FastAPI.
  • APIs REST y autenticación JWT.
  • Frontend (HTML/CSS/JS) y dashboards.
  • Despliegue en Linux.

Características

  • Monitoreo en tiempo real.
  • Gestión de usuarios y roles.
  • Alertas automáticas.

Tecnologías

Python · FastAPI · PostgreSQL · JavaScript · HTML5 · CSS3 · JWT · Linux
