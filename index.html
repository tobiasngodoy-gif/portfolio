Producto en desarrollo

Vision-AR

Vision-AR es una plataforma web de monitoreo de infraestructura tecnológica desarrollada para centralizar la supervisión de cámaras de videovigilancia, dispositivos de red y servidores desde una única interfaz.

Como fundador y desarrollador principal, participo junto al equipo en el diseño, desarrollo, implementación y evolución continua del producto.

Tecnologías

Python · FastAPI · PostgreSQL · JavaScript · HTML5 · CSS3 · JWT · Linux

Características resumidas

  • Monitoreo centralizado de dispositivos.
  • Gestión de usuarios y roles.
  • Dashboards administrativos.
  • Automatización de alertas.
  • Validación de conectividad y disponibilidad.
  • Arquitectura preparada para múltiples organizaciones.
  • Despliegue y mantenimiento sobre servidores Linux.
